Core ML.NET Techniques
	Time Series Forecasting:
		Uses the ForecastBySSA (Singular Spectrum Analysis) algorithm for univariate forecasting.
		Ideal for predicting future values based solely on historical order and time-sequenced data.
	Regression Analysis:
		Used to predict a continuous value (the stock price) based on multiple related features.
		Common algorithms include Online Gradient Descent (suited for real-time predictions) and 
		FastTree or LightGBM for more complex, non-linear relationships.
	Deep Learning:
		ML.NET supports integrating TensorFlow models, allowing for advanced architectures like 
		LSTM (Long Short-Term Memory) which are highly effective for sequence modeling in stock prices.
